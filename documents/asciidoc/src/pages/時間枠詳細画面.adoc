=== 時間枠詳細画面

==== 概要

[.lead]
時間枠の詳細情報を表示する。

[plantuml]
--
rectangle {
  database "時間枠" as input

  ("時間枠詳細画面") as (prog)
}

input --> prog : input
--

===== I/O

|======================================
| 名称 | 物理名 | 種類 | I/O種別 | 備考
| 時間枠 | time_indices | DB | I |
|======================================
