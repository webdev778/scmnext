=== 会社(PPS)詳細画面

==== 概要

[.lead]
会社(PPS)の詳細情報を表示する。

[plantuml]
--
rectangle {
  database "会社(PPS)" as input

  ("会社(PPS)詳細画面") as (prog)
}

input --> prog : input
--
